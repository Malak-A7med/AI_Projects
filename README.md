## Getting Started

### Prerequisites

```bash
python >= 3.8
pip >= 21.0
```

### Installation

1. **Clone the repository:**
```bash
git clone https://github.com/Malak-A7med/AI_Projects.git
cd AI_Projects
```

2. **Create virtual environment:**
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. **Install dependencies:**
```bash
pip install -r requirements.txt
```

4. **Explore projects:**
```bash
cd <project_folder>
jupyter notebook
```

## 📌 Tips for Contributing New Projects

- **Notebooks:** Include detailed comments and markdown explanations
- **Datasets:** Document data source, collection method, and preprocessing steps
- **Models:** Save trained models with version information and hyperparameters
- **README:** Include setup instructions, results summary, and key findings
- **Documentation:** Follow the project template structure for consistency
- **Code Quality:** Use meaningful variable names and add docstrings
- **Demo:** Consider adding interactive demos using Streamlit or Gradio
